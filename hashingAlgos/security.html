Security Algorithms

The security of a computer system is a crucial task. It is a process of ensuring the integrity of the OS.
A system is said to be secure if its resources are used and accessed as intended, but no system can guarantee absolute security from all malicious threats.

Security can be threatened via two violations:
- Threat: A program that can cause damage to the system.
- Attack: An attempt to break security and make unauthorized use of assets.

SHA1

SHA-1 produces a 160-bit hash value and is used in protocols like TLS, SSL, SSH. Though once widely used, it's now considered weak against attacks.

SHA256

SHA-256 produces a 256-bit hash value. It is highly secure and used in modern cryptographic applications and blockchain technology.

MD4

MD4 generates a 128-bit hash but is no longer secure due to its vulnerability to attacks. It is outdated and deprecated for security use.

MD5

MD5 is commonly used for checksums and file integrity, but is not cryptographically secure due to collision vulnerabilities.
